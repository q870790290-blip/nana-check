# Nana Check

Nana Check is a minimal local MVP for a pay-per-use analysis capability.

It accepts a target project, tool, protocol, or platform and returns a quick structured judgment including:

- verdict
- confidence
- score
- green flags
- red flags
- next checks

## Current stage

This is an early MVP focused on proving the core interaction loop:

1. user inputs a target
2. Nana Check analyzes the input
3. structured output is returned

## Local run

```bash
npm install
npm start
```

Then open:

```bash
http://127.0.0.1:3017
```

## Test

```bash
npm test
```

## Product direction

This project is intended to evolve toward a crypto-native, pay-per-use research and judgment capability.
